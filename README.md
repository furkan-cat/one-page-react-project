## About The Project
This is my first one page react project. Fully responsive for all devices. If you click the arrow button which is located behind the mini van you'll be smooth scrolled to the bottom of web site. Added some card animations and blur effects.

[Project-Demo](https://onepage-react.netlify.app/)


#### React Used

* DOM Manipulation
* Smooth Scrolling with React
* Youtube Vide Player
* React CSS Manipulation
* Css Animations

### Built With
* [React](https://www.javascript.com/)
* [HTML](https://html.com/)
* [CSS](https://css.com/)
